Работа с компаньоном Keypad Companion

Keypad Companion — это фоновое приложение Windows, которое позволяет вашему макропаду автоматически менять раскладки (профили) при переключении между различными приложениями.

Как использовать:

  1. Настройте привязки клавиш и крутилок для нужного приложения.
  2. Нажмите кнопку 💾 у соответствующего профиля в списке, чтобы применить текущую конфигурацию.
  3. Настройте правила (например, введите имя процесса Spotify.exe для профиля музыки).
  4. Нажмите кнопку «Сохранить в компаньон» для отправки данных.
  5. Запустите приложение Keypad_Companion.exe на вашем компьютере.

Преимущества компаньона:

  • Работает полностью автономно в фоновом режиме.
  • Не требует держать открытой вкладку браузера или иметь доступ к интернету.
  • Почти не расходует системные ресурсы компьютера (написан на Python/C, не грузит CPU).
  • Сам опрашивает активные окна Windows 1 раз в секунду и мгновенно перепрошивает макропад.
